EventViz

Interactive 3D visualization of Les Houches Event and HepMC files

What is EventViz?

EventViz is a lightweight, browser-based tool for exploring particle-collision events stored in Les Houches Event (LHE) or HepMC format. It reads event metadata and per-particle kinematics directly from your file and renders an interactive three-dimensional, transvere and longtidunal views of the collision: incoming beams, hard-scatter products, and decay chains with correct angular separation.

Visit the Github page for more information.

How it works

  1. Upload an uncompressed .lhe or .hepmc file using the panel on the left.
  2. Inspect generator, process, cross section, PDF, tune parameters, and beam information parsed from the file.
  3. Navigate events by number or with the arrow buttons — only the selected event is loaded for speed.
  4. Explore the 3D view: orbit with the mouse, zoom, and hover over particle tracks for full kinematic details.

Particle identities are resolved from PDG ID codes. Decay topology is reconstructed from mother–daughter indices in the event record, so decay products are drawn branching from their parent particle's endpoint.

Roadmap

Future releases may add histogramming of single-particle kinematics (pT, η, φ, energy, and mass, if available) and multiparticle variables (ΔR, invariant mass, MET, HT, etc.) for the final state particles.
